About

In 2016 I graduated from University with a BA in Creative Writing. Following this I bounced around jobs as I tried to set up several blogs to pursue a career in writing. I’d set up these sites using WordPress and, in the process, found that I enjoyed setting up the sites more than the writing itself.


Fast forward to October 2020 when I started to look into web development more seriously. After various courses and articles, I came across The Odin Project curriculum. I dived in and tried to work on it daily in tandem with my full-time job.


In June 2021 I felt like I was in a position to quit my job in Collections as I had found something that I wanted to pursue as a career. I am now seeking a job in web development whilst working on projects within the curriculum.


Outside of web development I enjoy football, going to gigs, and learning German.


Projects

Noughts and Crosses

Here I created a Tic-Tac-Toe game (or noughts and crosses) as part of The Odin Project Curriculum. The project used HTML, CSS, and Javascript. The game is played by one player against a computer that will make random moves until the game ends in either a player win, a draw, or a computer win.


Weather app

Here I created an app that would show the weather conditions for a location as part of the The Odin Project curriculum. The project used HTML, CSS, Javascript, Webpack, and OpenWeatherMap API. The main learning outcome from the project was to use the OpenWeatherMap API as well as the await keyword to create an app for the first time.


Library

Here I created a Library app to keep a list of books you've read/not read as part of The Odin Project Curriculum. I used HTML, CSS, and Javascript for this project and also made use of localstorage for the first time so that the list would save if you refreshed the page on the same device.


Battleship

Here I created a Battleship game as part of The Odin Project Curriculum that can be played against the computer using Javascript, HTML, CSS, Webpack, and Jest. The main learning point for this project was to use Jest to test each module that didn't involve DOM manipulation to ensure that unit worked instead of using the console.
